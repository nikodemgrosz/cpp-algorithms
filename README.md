# Advanced C++ Algorithms & Performance Optimization 

## Files

* `tarzan.cpp` (TSP Solver):** Searching the best way to reach using dynamic programming with bitmasks.
* `paczki.cpp` (Knapsack Problem):** Finding the best profit using recursion.
* `przekatne.cpp` (Graph Cycle Detection):** Counting 4-length cycles using combinatorics and adjacency list.
* `mobius.cpp` (Graph Problem):** Based on own idea with arms and sticks.
* `pp.cpp`:** Math and geometry exercises.

## Optimizations
* **Fast I/O:** Custom input parsing instead of slow `std::cin`.
* **Memory Management:** Using `.reserve()` to prevent expensive heap allocations during loops.

## Technologies
`std::vector`, `std::string`, `std::cmath`.
